# McDonald's Nutrition Menu 🍟

An interactive, responsive, and visual McDonald's menu that displays nutritional information, calories, and prices for various items, powered by the unofficial McDonald's MCP Server.

## Features
- **Nutritional Information**: Detailed breakdown of protein, fat, carbohydrates, sodium, calcium, and energy.
- **Categorization**: Easily filter by Breakfast, Burgers, Snacks, Drinks, Desserts, and Coffee.
- **Calorie Badges**: Quick glance at the calorie count for each item.
- **Nutrition Table**: A comprehensive reference table for all items at the bottom of the page.
- **Responsive Design**: Works great on both desktop and mobile devices.

## How to Run

1. Clone the repository.
2. Start a local HTTP server in the directory:
   ```bash
   python -m http.server 8080
   ```
3. Open `http://localhost:8080/mcdonalds_menu.html` in your browser.
